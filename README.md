# fluid-cdss
本ツールは、水分管理に関する臨床判断を支援するための教育用CDSS（Clinical Decision Support System）である。  患者の基本情報、バイタルサイン、検査データ（Na、BUN、Creなど）を入力することで、血清浸透圧（Posm）、BUN/Cre比、eGFRを自動計算し、脱水タイプ（高張性・等張性・低張性）を推定する。  さらに、ショックや重度電解質異常などのCritical条件を自動検出し、臨床上の注意点や判断理由を表示することで、看護師の思考過程を可視化することを目的としている。  本ツールは教育・研究用途を想定しており、診断や治療の決定を代替するものではない。
